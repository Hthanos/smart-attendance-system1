# Documentation Assets

This directory contains images, diagrams, and other assets used in the documentation.

## Contents

- **screenshots/** - Application screenshots
- **diagrams/** - Architecture and flow diagrams
- **logos/** - Institution and project logos
- **mockups/** - UI/UX mockups

## Adding Assets

When adding new assets:
1. Use descriptive filenames (e.g., `dashboard-screenshot.png`)
2. Optimize images for web (compressed)
3. Preferred formats: PNG for screenshots, SVG for diagrams
4. Update this README with descriptions

## Usage in Documentation

Reference assets using relative paths:
```markdown
![Dashboard Screenshot](assets/screenshots/dashboard.png)
```
